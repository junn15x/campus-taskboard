
Campus Taskboard API
 We are creating a REST API with proper HTTP

How to Run:
Require Java 17 or 21 

run  CampusTaskboardApplication.java

Endpoints

1.	GET all task
METHOD: GET
HEADER: NONE

2.	POST Create a Task
METHOD: POST
HEADER: CONTENT-TYPE : APPLICATION/JSON

3.	GET TASK by ID
METHOD: GET

4.	PUT update a task
METHOD: PUT
HEADER: CONTENT-TYPE: APPLICATION/JSON

5.	DELETE a task
METHOD: DELETE

6.	Test Validation
METHOD: POST
CONTENT-TYPE: APPLICATION/JSON
